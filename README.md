# Portifolio Samuel R. Caroba

Este repositório contém o código-fonte do meu portfólio web pessoal. Você pode visualizá-lo online em: [https://portifolio-eta-bay-60.vercel.app/](https://portifolio-eta-bay-60.vercel.app/)

## 🚀 Conteúdo
O portfólio foi construído com foco em performance e design moderno, utilizando:
*   **HTML5** e **Tailwind CSS** para a estrutura e estilização.
*   **Lucide Icons** para ícones vetoriais.
*   **AOS (Animate On Scroll)** para animações fluidas.
*   **JavaScript** para interatividade (menu mobile).
