# Portifolio Samuel R. Caroba

Este repositório contém o código-fonte do meu portfólio web pessoal, criado para demonstrar minhas habilidades em desenvolvimento Full-Stack.

## 🚀 Conteúdo
O portfólio foi construído com foco em performance e design moderno, utilizando:
*   **HTML5** e **Tailwind CSS** para a estrutura e estilização.
*   **Lucide Icons** para ícones vetoriais.
*   **AOS (Animate On Scroll)** para animações fluidas.
*   **JavaScript** para interatividade (menu mobile).

## 🔗 Projetos Destacados
A página apresenta os principais projetos disponíveis no meu GitHub:

1.  **ENEM Speedrun**: Plataforma educacional Full-Stack.
2.  **Caroba Contabilidade**: Sistema web para serviços contábeis.
3.  **Taskflow**: Gerenciador de tarefas para equipes.
